# superTrunfoMestre.c
jogo do super trunfo nível mestre
tutorial:
1 - escolha entre dois países 
2 - a janela mostrará 3 características de cada um dos países
3 - selecione 2 características a seu critério, elas serão utilizadas para comparação entre as outras características do outro país
4 - aprecie o resultado!
Bom jogo!
